***DevCpp + NEO SDK安装文档***
================================================
首先是下载DEV-CPP，地址是：http://www.8623.com/tool/DEVCPP.rar

里面有一个自解压包：devall4113.exe和一个安装程序devcpp4990.exe

一、安装：
　　1.首先执行附带的Devcpp4980.EXE安装DEV-CPP多国语言版，安装时记得选简体中文（Chinese）；
　　2.然后将devall4113.exe包里的INCLUDE和LIB都复制到DEVCPP的安装目录下相应的子目录（注意：dll目录里的东
    西要放在WINDOWS\SYSTEM目录下）。

二、相关设置：
　　在编译时，打开“工具”（OPTIONS）菜单选择“编译选项”（COMPILER OPTIONS）
　　在“编译时加入以下命令”（ADD THE FOLLOWING COMMANDS WHEN CALLING COMPILER）下方的文本框里输入：
    -lalleg
　　如果是工程，就在工程里加入这个编译参数。
    重启Dev-cpp主程序使设置生效。

三、将这个包中的NEO.H拷贝到DEV-CPP的include文件夹中。

一切就绪之后你就可以用DEVCPP + NEO开发程序了（试试用DEV编译附带的ex03.cpp）
目前的NEO for DEV版不是很完善，像光标文件的显示就没有加入，我会在近期将其做完

最后有两点需要注意的：用DEV + NEO写程序源代码必须以cpp做扩展名，也就是必须是C++
文件，不会C++的也不用但心，直接在C++文件里写C语言内容不犯法。再就是必须在main()函数末尾加上END_OF_MAIN()，
void main()
{
   /* ... */
}
END_OF_MAIN()

否则编译器会报“undefined reference to `WinMain@16' ld returned 1 exit status ”
