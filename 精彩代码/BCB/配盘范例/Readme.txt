      《C++Builder与Windows API经典范例》配书光盘自述文件

1.光盘内容
  Readme.txt  光盘说明文件
  \配盘范例  《C++Builder与Windows API经典范例》书中各章范例所在
的文件夹，包含完整的工程范例程序代码。6个文件夹具体内容如下：
  Ch1_File Functions为文件函数，包括41个与文件操作相关的函数范例。
  Ch2_Bitmap Functions为位图函数，包括37个与图像操作相关的函数范例。
  Ch3_Window Functions为窗口函数，包括93个与窗口操作相关的函数范例。
  Ch4_Keyboard Input Functions为键盘输入函数，包括23个与键盘输入操
作相关的函数范例。
  Ch5_Mouse Input Functions为鼠标输入函数，包括12个与鼠标输入操作相
关的函数范例。
  Ch6_Hook(Keyboard & Mouse)为Hook实例，包括具有完整注解的鼠标和键
盘Hook连接文件(DLL)与执行文件(EXE)程序代码。

2.光盘运行环境
  本光盘可以运行在Windows 操作系统下，包括98/2000/NT/XP。
  前5章范例程序的软件环境是C++ Builder 6.0,附录中(即Ch6_Hook文件夹
中)有的范例运行环境为C++ Builder 5.0,有的则为C++ Builder 3.0。

3.光盘使用说明
  在使用本光盘的范例文件时，要先把“\配盘范例”文件夹下的内容复制
到硬盘，然后把各章范例文件的只读属性取消。本光盘中目录与书中的章
节，相互契合，可互为索引参考；光盘中目录的命名说明如下：      
  "Ch1_File Functions"目录，即范例集的"第1章 File Functions"，依此
类推。
  目录中各子目录(如"ch01-06-2"子目录)，即范例集的"第1章中，第6个函
数的第2个范例"，依此类推。
